Exe6 Pseudo Plan

Login Page
  1. If user already logged in, redirect to catlog page;
  2. Otherwise, display empty login form;
  3. When submit login form, check validity;
  4. If any field login invalid (empty), display the form with submitted data, and error messages;
  5. If all required field are filled with value, send sql query to database to find customer;
  6. If customer not found, display error message;
  7. If login succeeded, redirect to catalog page.
Registration Page
  1. When registering, display empty registration form;
  2. When submit registration form, check validity;
  3. If any field login invalid (empty), display the form with submitted data, and error messages;
  4. If all required field are filled with value, send sql query to database to create customer;
  5. If any error, display error message;
  6. If customer is created, display success message;
Catalog Page
  1. If user didn't login, redirect to login page;
  2. Fetch and display all categories;
  3. Display products of selected category, if user selected one
  4. Add a product to the cart associated with the user, when user clicked "Buy" button;
Shopping Cart Page
  1. If user didn't login, redirect to login page;
  2. Display empty message if no user haven't buy any products;
  3. If there's any product in cart, fetch and display products;
  4. Calculate and display total cost;
