Exe6 Pseudo Plan

Login Page
  1. When login, display empty login form;
  2. When submit login form, check validity;
  3. If any field login invalid (empty), display the form with submitted data, and error messages;
  4. If all required field are filled with value, send sql query to database to find customer;
  5. If customer not found, display error message;
  6. Redirect to catalog page.
Registration Page
  1. When registering, display empty registration form;
  2. When submit registration form, check validity;
  3. If any field login invalid (empty), display the form with submitted data, and error messages;
  4. If all required field are filled with value, send sql query to database to create customer;
  5. If any error, display error message;
  6. If customer is created, display success message;
Catalog Page
  1. Fetch and display all categories;
  2. Display products of selected category, if user selected one
  3. Add a product to the cart associated with the user, when user clicked "Buy" button;
Shopping Cart Page
  1. Display empty message if no user haven't buy any products;
  2. If there's any product in cart, fetch and display products;
  3. Calculate and display total cost;
